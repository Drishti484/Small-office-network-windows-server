# Small-office-network-windows-server
Small office network design and implementation using windows server with Domain controller and DNS Configuration

## Project Overview
This Project demonstrates the implementation of a small enterprise network using Windows server as Domain Controller and DNS Server.

## Network Architecture
- 1 Domain Controller (Windows Server)
- 2 Client Machines
- LAN Network using VirtualBox

## Technologies Used
- TCP/IP
- DNS
- Active Directory
- Windows Server
- VirtualBox

## IP Configuration
Server: 192.168.1.10
Client1: 192.168.1.11
Client2: 192.168.1.12

## Testing Commands
-ping
-ipconfig
-nslookup

## Outcome
Successful domain join, DNS resolution and communication between clients and server.
